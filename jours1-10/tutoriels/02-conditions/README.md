# Tutoriel – Conditions en PHP

## Objectif
Faire des choix avec `if`, `else`.

## Code

```php
<?php
$heure = 14;

if ($heure < 12) {
    echo "Bonjour du matin !";
} else {
    echo "Bonne après-midi !";
}
?>
```

## Explication
- Condition selon l’heure
- Affiche un message différent selon la valeur
